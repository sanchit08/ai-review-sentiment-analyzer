# AI Review Sentiment Analyzer

This project analyzes product reviews using Large Language Models.

Features:

- Extract reviews from PDF files
- Perform sentiment analysis
- Identify key product themes
- Extract pros and cons
- Generate structured outputs

Tech Stack:

- LangChain
- OpenAI GPT Models
- Python
- PDF Processing

Output:

The system generates structured results including:
- Sentiment
- Summary
- Key themes
- Pros
- Cons

Results are saved in JSON and CSV format.
